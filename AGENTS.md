# Agent Instructions

This repository uses the Cursor rules entry at [`.cursor/rules/index.md`](.cursor/rules/index.md) as the single source of truth for AI and agent behavior.

Before planning, editing, reviewing, or generating code, read and follow:

- [`.cursor/README.md`](.cursor/README.md) for the Cursor configuration overview
- [`.cursor/rules/index.md`](.cursor/rules/index.md) for the actual project rules and referenced sub-rules

If any local guidance file conflicts with the Cursor rules entry or its referenced rule files, the Cursor rules entry wins.
