# Frontend Best Practices / 前端最佳实践

1. **Component Single Responsibility / 组件单一职责**: Each component does one thing.
2. **Type Safety / 类型安全**: Fully utilize TypeScript.
3. **Performance Optimization / 性能优化**: Avoid unnecessary re-renders.
4. **Accessibility / 可访问性**: Follow a11y standards.
5. **Test Coverage / 测试覆盖**: Write tests for critical components.
6. **Internationalization / 国际化**: Always add translations to all locale files.
