# 报表生成器插件 (Report Generator Plugin)

基于自然语言提示词，使用系统数据生成报表的插件。

## 功能特性

- 📝 **自然语言输入**：通过提示词描述需要生成的报表
- 🔍 **智能识别**：自动从提示词中识别数据集合
- 📊 **Excel报表**：生成标准Excel格式的报表文件
- 🌐 **多语言支持**：支持中文和英文界面

## 使用方法

1. 在系统设置中找到"报表生成器"
2. 选择数据集合（可选，留空将自动识别）
3. 输入提示词，例如：
   - "生成用户列表报表"
   - "生成上个月的所有订单报表"
   - "生成用户列表报表"
4. 点击"生成报表"按钮
5. 报表将自动下载为Excel文件

## 技术实现

### 服务端

- 接收提示词和参数
- 解析提示词，识别数据集合
- 查询系统数据
- 生成Excel报表并返回

### 客户端

- 提供友好的用户界面
- 支持数据集合选择
- 显示可用字段信息
- 处理文件下载

## 开发

# 构建插件
